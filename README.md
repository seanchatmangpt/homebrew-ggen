# Seanchatmangpt Ggen

## How do I install these formulae?

`brew install seanchatmangpt/ggen/<formula>`

Or `brew tap seanchatmangpt/ggen` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "seanchatmangpt/ggen"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
